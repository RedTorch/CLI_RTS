GENERAL COMMAND FORMAT:

In this game, everything is controlled through text commands which follow a specific format:
target action param1 param2 param3
target: the selected object
action: the action you wish the target to perform
param1-3: depending on the action, parameters may be required.

OBJECTS:

  • "dev": a debugging object not meant for the final game.

    • "spawnHostile [x]": spawns x many hostiles randomly across the map.
    • "spawnDrone [name] [x] [y]": spawns a single drone with name [name] at the coordinates (x,y).

  • "[name of drone]": a player-controlled drone.

    • "move [x] [y]": directs the drone to move to the location (x,y).
    • "move [droneName]": directs the drone to move to the current location of the drone "droneName".
    • "stop": stops the drone. This will also erase stored movement coordinates.
    • "rename [new name] ": renames the drone to [new name]. Note that this name is used to control the drone as well.

Example:

"drone1 move 300 400"
^ drone1 moves to (300,400)

MAP:

The game map is 1200px x 800px (width x height). Note that when specifying coordinates, the y-value is the distance from the top of the screen, not the bottom. For example, (300,700) would be near the bottom of the screen, while (200,100) is near the top.
